# 📊 Power BI Dashboard

## 🔹 Overview

This folder contains the Power BI report file used to visualize the processed sales data.

---

## 📂 File

* **Sales_Analysis_Dashboard.pbix**

  * Power BI dashboard file
  * Built using processed data from Azure Databricks

---

## 📈 Dashboard Features

The dashboard provides interactive insights including:

* **Category-wise Sales**

  * Displays total sales grouped by product category

* **Region-wise Orders**

  * Shows number of orders across different regions

* **Key Performance Indicators (KPIs)**

  * Total Sales
  * Total Orders
  * Performance metrics

---

## 📊 Visualizations Used

* Bar Charts (Category-wise sales)
* Column Charts (Region-wise orders)
* KPI Cards (Total Sales, Orders)
* Filters/Slicers for dynamic analysis

---

## 🔄 Data Source

* Data is processed using **Azure Databricks (PySpark)**
* Stored in **Azure Data Lake (Delta format)**
* Connected to Power BI for visualization

---

## 🧠 Purpose

This dashboard helps in:

* Understanding sales trends
* Identifying high-performing categories
* Analyzing regional performance

---

## 🚀 Notes

* Open `.pbix` file using Power BI Desktop
* Ensure data source connections are configured properly
