# Sales Analysis Dashboard (End-to-End Data Engineering Project)

## 📌 Project Overview

This project demonstrates an end-to-end data pipeline using Azure Data Factory, Databricks, and Power BI to analyze sales data and generate business insights.

## ⚙️ Tech Stack

* Azure Data Factory (ADF)
* Azure Data Lake Storage (ADLS)
* Azure Databricks (PySpark & SQL)
* Power BI

## 🔄 Data Pipeline Workflow

1. Data is ingested from source using Azure Data Factory
2. Stored in Azure Data Lake Storage
3. Processed and transformed using Databricks (PySpark)
4. Data is stored in Delta format
5. SQL queries are used for aggregation
6. Power BI dashboard visualizes insights

## 📊 Key Features

* Category-wise sales analysis
* Region-wise order count
* Total sales aggregation
* Interactive dashboard

## 📁 Project Structure

* ADF → Pipelines & datasets
* Databricks → Notebooks (PySpark + SQL)
* PowerBI → Dashboard file (.pbix)

## 🚀 Key Learnings

* Built ETL pipeline using ADF
* Processed data using PySpark
* Created Delta tables
* Built Power BI dashboard
